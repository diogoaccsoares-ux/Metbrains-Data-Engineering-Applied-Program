# Data Engineering Applied Program Assignments

This repository contains a collection of assignments showcasing data engineering, Python, Databricks, and SQL skills. Each part highlights a practical project involving data extraction, transformation, analysis, and visualization.

---

## Part 1: Google Colab Data Engineering Assignment
Built a pipeline that extracts messy text data, cleans it, transforms it into a structured format, loads it into a local SQL database, and uses GenAI to enrich the data and auto-generate documentation.

---

## Part 2: Databricks Assignment (Basics + Spark DataFrames)
Processed a CSV in Databricks using PySpark: explored, filtered, aggregated, and transformed data with new columns and missing value handling. Saved the results as a Delta table and ran SQL queries for analysis.

---

## Part 3: Python Assignment (Basics + Data Handling)
Explored and transformed a CSV dataset with pandas: handled missing values, filtered rows, created new columns, and computed summary statistics. Visualized key features with histograms and bar charts, then saved the cleaned dataset to CSV.

---

## Part 4: SQL Assignment (Beginner + Analytics Focus)
Analyzed `customers` and `orders` tables using SQL: retrieved specific customers, counted and aggregated orders, and identified customers with no orders. Calculated totals, tracked monthly sales trends, and categorized orders by amount into Low, Medium, and High.

---

## Technologies & Tools
- Python, Pandas
- PySpark, Databricks
- SQL (SQLite/MySQL/PostgreSQL)
- Google Colab
- Data visualization: Matplotlib, Seaborn
- Delta Lake
- GenAI for automated enrichment & documentation
